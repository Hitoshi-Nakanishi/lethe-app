# Lethe 使い方ガイド

English version: [usage.md](usage.md)

## 基本フロー

```text
録音 -> 文字起こし -> メモに用語を記入 -> メモで校正 -> 議事録を作成
```

## 録音する

1. マイク入力と PC 出力のセレクタで使うデバイスを選びます。
2. Windows では **PC音声を取る** が既定でオンになり、YouTube、Zoom、ブラウザ、ローカルアプリなど PC から再生されている音を録音します。
3. マイク音声を録らない場合は **マイク音声を取る** をオフにします。**PC音声を取る** がオンなら、PC 出力だけを録音・文字起こしできます。
4. 部屋にファンや空調のような定常ノイズがある場合はノイズ除去を有効にします。
5. **録音** をクリックするか `Space` を押します。
6. 音声が流れている間、メーターの Mix、Mic、PC レベルが反応していることを確認します。
7. **停止** で録音を終了します。

ライブ文字起こしを有効にしている場合は、録音中に暫定結果が表示されます。録音後は高精度文字起こしを実行して、タイムスタンプ付きの最終結果にします。

Whisper モデルが未導入の場合、Lethe は録音停止時に自動ダウンロードを開始しません。音声だけを保存できる状態にしておき、あとでモデルを入れてから解析できます。

## 文字起こしする

録音後、または既存の音声ファイルを開いた後に高精度文字起こしボタンを押します。Lethe は Whisper で音声を解析し、1 セグメント 1 行のタイムスタンプ付き文字起こしを表示します。`[02:15]` のようなタイムスタンプをクリックすると、その位置から音声を再生できます。文字起こしや校正などの解析中は、同じメーターが解析 wave と進捗を表示します。

録音だけ保存したセッションや音声ファイルは、あとでアプリの **音声を開く** または **セッションを開く** から読み込めます。モデル導入済みなら解析が走り、未導入なら **高精度で文字起こし** を押したときにダウンロード確認が出ます。

## メモを書く

右側のメモ欄に固有名詞、人名、製品名、専門用語を書きます。1 行 1 語にすると確認しやすくなります。メモはライブ文字起こし時の Whisper コンテキストと、校正時の正しい表記として使われます。

## メモで校正する

文字起こしとメモが揃ったら校正ボタンを押します。Ollama がメモの表記を優先し、意味、順序、タイムスタンプを保ったまま誤認識らしい箇所を修正します。

校正や議事録生成の前に **LLM** セレクタから使うモデルを選べます。候補を増減する場合は `default.toml` を編集します。

## 議事録を作成する

議事録ボタンを押すと、文字起こしから Markdown の議事録を生成します。結果は別ウィンドウで開き、編集して `.md` として保存できます。

## セッション

ファイルメニューからセッションを保存、または開けます。セッション `.zip` には次のファイルが入ります。

```text
audio.wav
transcript.txt
notes.txt
meta.json
```

## 保存先

保存と読み込みダイアログの共通フォルダは `default.toml` の `[paths].datasets_dir` で指定できます。空のままにすると OS の標準動作に任せます。

MP3 保存ダイアログに表示される推奨ファイル名は `default.toml` の `[filenames]` で指定できます。標準では `YYYYMMDD_HHMM_<meeting_name>.mp3` 形式になり、保存ダイアログ上でそのまま編集できます。

## テーマ

ヘッダーのテーマセレクタから Midnight、Aurora、Ember を選べます。**Dark** トグルで各テーマのライト版とダーク版を切り替えます。

## 言語

ヘッダーの言語セレクタから UI を日本語と英語で切り替えできます。選択した言語は保存され、次回起動時にも復元されます。

## キーボードショートカット

| キー | 動作 |
|---|---|
| `Space` | テキスト欄にフォーカスがないとき、録音開始または停止 |
| `Cmd/Ctrl+S` | 文字起こしを保存 |
| `Cmd/Ctrl+O` | 音声ファイルを開く |
| `Cmd/Ctrl++` | アプリのフォントサイズを大きくする |
| `Cmd/Ctrl+-` | アプリのフォントサイズを小さくする |
| `Cmd/Ctrl+Z` | テキスト編集を取り消し |

## 精度を上げるコツ

- できるだけ話者の近くで録音します。
- 名前や専門用語は録音前または録音中にメモへ入れます。
- 怪しい行はタイムスタンプをクリックして聞き直します。
- 速度を優先したい場合は Whisper モデルを小さくします。
